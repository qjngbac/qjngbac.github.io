手机屏幕技术深度解析：以OLED为例

发表于2026-07-05|更新于2026-07-08|技术科普
|浏览量:
你每天看手机屏幕超过 6 小时，但你真的了解它吗？

一、屏幕基础概念

1.1 像素与分辨率

屏幕是由无数个像素（Pixel）组成的，每个像素可以独立发光或显示颜色。

术语 含义 示例
分辨率 屏幕像素的总数 2778×1284（iPhone 14 Pro Max）
PPI 每英寸像素数，数值越高越细腻 460 PPI
FHD+ 全高清升级版 2400×1080
2K 超高清 3200×1440

PPI 的计算公式：

PPI = √(水平像素² + 垂直像素²) / 屏幕尺寸(英寸)

人眼在 25cm 距离下，大约能分辨 300 PPI 以上的细节。所以现在主流手机的 400+ PPI 已经足够清晰。

1.2 色域与色准

色域（Color Gamut）是屏幕能显示的颜色范围：

色域标准 说明 应用
sRGB 最基础的标准，覆盖约 35% 可见色 网页、日常使用
DCI-P3 影视级色域，比 sRGB 宽 25% 旗舰手机、影院
Rec.2020 超广色域，覆盖约 75% 可见色 8K 电视、专业显示

色准用 ΔE（Delta E）衡量：

  • ΔE < 1：人眼无法分辨差异，专业级
  • ΔE < 3：普通人看不出差别
  • ΔE > 5：明显偏色

1.3 刷新率与触控采样率

参数 含义 主流值
刷新率 每秒画面更新次数 60Hz / 90Hz / 120Hz / 144Hz
触控采样率 每秒检测触控的次数 240Hz / 480Hz / 720Hz

刷新率越高，动画越流畅。120Hz 比 60Hz 流体感提升明显，但也更耗电。LTPO 技术可以动态调节刷新率（1Hz~120Hz），兼顾流畅和续航。

二、OLED 的工作原理

2.1 什么是 OLED

OLED（Organic Light-Emitting Diode）= 有机发光二极管。

核心特点：自发光。每个像素自己就是光源，不需要背光模组。这和 LCD 完全不同——LCD 需要一整块背光板，再通过液晶分子控制光的通过量。

LCD 结构（从下到上）：
┌─────────────────────┐
│      偏光片          │
├─────────────────────┤
│      彩色滤光片      │
├─────────────────────┤
│      液晶层          │ ← 控制光的通过
├─────────────────────┤
│      背光模组        │ ← 整块发光
└─────────────────────┘

OLED 结构（从下到上）：
┌─────────────────────┐
│      封装层          │
├─────────────────────┤
│   有机发光层         │ ← 每个像素独立发光
├─────────────────────┤
│      TFT 驱动层     │
└─────────────────────┘

2.2 OLED 的发光原理

OLED 的核心是有机材料层，通电后电子和空穴在有机层中复合，释放能量以光的形式发出。

一个 OLED 像素的典型结构：

阴极（金属）
   ↓
电子传输层（ETL）
   ↓
发光层（EML）  ← 红/绿/蓝有机材料
   ↓
空穴传输层（HTL）
   ↓
阳极（ITO 透明导电层）
   ↓
基板（玻璃/塑料）

不同有机材料发出不同颜色的光：

  • 红色：磷光铱配合物
  • 绿色：磷光铱配合物（效率最高）
  • 蓝色：荧光材料（寿命最短，是技术瓶颈）

2.3 AMOLED vs PMOLED

类型 驱动方式 应用
PMOLED（被动矩阵） 逐行扫描，简单但低效 小型设备、手环
AMOLED（主动矩阵） 每个像素有独立 TFT 驱动 手机、电视

手机上说的 OLED 其实都是 AMOLED。

三、OLED 的关键参数

3.1 亮度

参数 含义 旗舰水平
典型亮度 日常使用的亮度 800~1200 nits
峰值亮度 局部高光瞬间的最高亮度 2000~3000 nits
全屏亮度 整个屏幕白色时的亮度 1000~1500 nits

为什么峰值亮度比全屏亮度高？

因为 OLED 每个像素独立供电，局部高亮时电流集中，散热可控。全屏高亮时功耗和发热都会成为瓶颈。

3.2 PWM 调光

OLED 调节亮度有两种方式：

方式 原理 优缺点
DC 调光 直接改变电流大小 低亮度时色偏严重
PWM 调光 高频开关屏幕，调节亮/灭比例 低亮度可能频闪伤眼

PWM 频率：

  • 低频 PWM（240Hz 以下）：敏感人群可能感到不适
  • 高频 PWM（1920Hz+）：基本无感知，目前旗舰主流
  • 类 DC 调光：软件模拟，降低频闪但牺牲色准

3.3 LTPO 技术

LTPO（Low-Temperature Polycrystalline Oxide）是一种背板技术，可以让刷新率在 1Hz~120Hz 之间动态切换。

场景 → 刷新率：
滑动列表 → 120Hz（流畅）
看静态图片 → 1Hz（省电）
播放视频 → 匹配帧率（24/30/60Hz）
息屏显示 → 1Hz（极低功耗）

LTPO 版本演进：

  • LTPO 1.0：仅部分 App 适配
  • LTPO 2.0：系统级自适应
  • LTPO 3.0：全场景无感切换

3.4 排列方式

OLED 子像素的排列方式直接影响清晰度：

排列 特点 代表
RGB Stripe 标准排列，最清晰 LCD 屏幕
Pentile（钻石排列） 绿色像素多，红蓝共享 三星 AMOLED
类钻排列 各厂商变体 京东方、维信诺
Delta 排列 三角形排列 少数 OLED 面板

为什么 Pentile 排列争议大？

同样 1080P 分辨率，Pentile 的实际等效清晰度通常低于 RGB Stripe，常被粗略估为 70%~80%，因为红蓝像素是相邻像素共享的。这也是为什么 OLED 屏需要更高标称分辨率才能达到 LCD 的清晰度。

四、OLED vs LCD

对比项 OLED LCD
发光方式 自发光 需要背光
黑色表现 纯黑（像素关闭） 灰黑（背光漏光）
对比度 无穷大 1500:1 ~ 5000:1
厚度 更薄（无背光层） 较厚
可弯折 可以（柔性基板） 不行
功耗 显示黑色时极省电 背光常亮，功耗固定
寿命 蓝色有机材料衰减快 相对更长
烧屏风险 有（长时间显示固定画面） 几乎没有
成本 较高 较低

五、OLED 的技术分支

5.1 柔性 OLED（Flexible OLED）

基板从玻璃换成聚酰亚胺（PI）塑料，可以弯折、卷曲。

应用：

  • 曲面屏（三星 Galaxy Edge 系列）
  • 折叠屏（Galaxy Z Fold、华为 Mate X）
  • 卷轴屏（概念产品）

5.2 Micro OLED / Micro LED

技术 特点 应用
Micro OLED 硅基 OLED（OLED-on-Silicon），超高 PPI Apple Vision Pro
Micro LED 无机材料，不烧屏，超亮 未来方向，成本极高

Micro LED 是业界公认的"终极显示技术"，但目前量产困难，成本是 OLED 的 10 倍以上。

六、屏幕供应商

供应商 特点 客户
三星显示（SDC） 技术最成熟，良率最高 苹果、三星、小米
LG Display（LGD） 大尺寸 OLED 强 苹果、谷歌
京东方（BOE） 国产第一，追赶中 华为、小米、荣耀
维信诺（Visionox） 折叠屏技术领先 华为、荣耀
天马微电子（Tianma） 中低端市场 OPPO、vivo
华星光电（CSOT） TCL 旗下，快速崛起 小米、三星

"屏幕混用"现象：同一款手机可能用不同供应商的屏幕，素质略有差异。这也是为什么网上经常讨论"屏幕抽奖"。

七、选购建议

7.1 关键参数优先级

亮度（户外可读性）> 色准 > 分辨率 > 刷新率 > 调光方式

7.2 不同需求的侧重点

需求 重点关注
户外使用 峰值亮度 > 2000 nits
护眼 高频 PWM（1920Hz+）或 DC 调光
看视频 DCI-P3 色域、HDR10+ / Dolby Vision
游戏 120Hz+、低触控延迟
折叠屏 关注折痕、UTG 玻璃厚度

八、常见问题

Q：OLED 烧屏怎么办？

A：避免长时间显示固定画面（如导航栏常亮），使用深色模式，开启自动亮度。现代 OLED 有像素偏移和补偿算法，正常使用 3 年内不必担心。

Q：PWM 调光真的伤眼吗？

A：因人而异。敏感人群在低亮度下可能感到不适。建议选择高频 PWM（1920Hz+）的手机，或在低亮度环境开启护眼模式。

Q：2K 屏幕比 1080P 耗电多少？

A：大约多 10%~20%。LTPO 技术可以部分弥补。日常使用建议 1080P+120Hz，需要时切 2K。
