手机屏幕技术深度解析：以OLED为例

发表于2026-07-05|更新于2026-07-19|技术科普
|浏览量:
你每天看手机屏幕超过 6 小时，但你真的了解它吗？

一、屏幕基础概念

1.1 像素与分辨率

屏幕是由无数个像素（Pixel）组成的，每个像素可以独立发光或显示颜色。

术语 含义 示例
分辨率 水平与垂直方向的像素数量 2796×1290（iPhone 14 Pro Max）
PPI 每英寸像素数，数值越高越细腻 460 PPI
FHD+ 非统一标准，通常指纵向像素多于 1920 的 1080P 屏 2400×1080
QHD / QHD+ 约 2560×1440 或更长比例，手机宣传中常称“2K” 3200×1440（QHD+ 示例）

PPI 的计算公式：

PPI = √(水平像素² + 垂直像素²) / 屏幕尺寸(英寸)

PPI 是否“足够”与观看距离、视力、子像素排列和显示内容有关。300 PPI 常被当作印刷或较远距离观看的经验参考，但在约 25cm 的手机观看距离下，部分用户仍能分辨更高像素密度带来的差异；因此不能只凭一个 PPI 数字判断清晰度。

1.2 色域与色准

色域（Color Gamut）是屏幕能显示的颜色范围：

色域标准 说明 应用
sRGB 常见网络与消费级内容标准 网页、日常使用
DCI-P3 / Display P3 比 sRGB 覆盖更宽的红、绿色范围 影院、手机与创作设备
Rec.2020 面向超高清电视的宽色域标准 UHD/HDR 内容与专业制作

色准用 ΔE（Delta E）衡量：

ΔE 越小通常表示实测颜色越接近参考值，但结论必须同时说明采用的公式（如 ΔE2000）、测试色样、亮度和白点。笼统地把“ΔE < 1”写成所有人都无法分辨并不严谨；选购时应比较同一测试方法下的平均值、最大值与灰阶表现。

1.3 刷新率与触控采样率

参数 含义 主流值
刷新率 每秒画面更新次数 60Hz / 90Hz / 120Hz / 144Hz
触控采样率 每秒检测触控的次数 240Hz / 480Hz / 720Hz

刷新率越高，动画通常越流畅，但面板、显示驱动与应用渲染也会共同影响功耗。LTPO 背板有利于扩大可变刷新率范围；具体能否降到 1Hz、支持哪些档位，仍取决于面板和整机方案。

二、OLED 的工作原理

2.1 什么是 OLED

OLED（Organic Light-Emitting Diode）= 有机发光二极管。

核心特点：自发光。每个像素自己就是光源，不需要背光模组。这和 LCD 完全不同——LCD 需要一整块背光板，再通过液晶分子控制光的通过量。

LCD 结构（从下到上）：
┌─────────────────────┐
│      偏光片          │
├─────────────────────┤
│      彩色滤光片      │
├─────────────────────┤
│      液晶层          │ ← 控制光的通过
├─────────────────────┤
│      背光模组        │ ← 整块发光
└─────────────────────┘

OLED 结构（从下到上）：
┌─────────────────────┐
│      封装层          │
├─────────────────────┤
│   有机发光层         │ ← 每个像素独立发光
├─────────────────────┤
│      TFT 驱动层     │
└─────────────────────┘

2.2 OLED 的发光原理

OLED 的核心是有机材料层，通电后电子和空穴在有机层中复合，释放能量以光的形式发出。

一个 OLED 像素的典型结构：

阴极（金属）
   ↓
电子传输层（ETL）
   ↓
发光层（EML）  ← 红/绿/蓝有机材料
   ↓
空穴传输层（HTL）
   ↓
阳极（ITO 透明导电层）
   ↓
基板（玻璃/塑料）

不同有机材料发出不同颜色的光：

  • 红色：磷光铱配合物
  • 绿色：磷光铱配合物（效率最高）
  • 蓝色：荧光材料（寿命最短，是技术瓶颈）

2.3 AMOLED vs PMOLED

类型 驱动方式 应用
PMOLED（被动矩阵） 逐行扫描，简单但低效 小型设备、手环
AMOLED（主动矩阵） 每个像素有独立 TFT 驱动 手机、电视

当前智能手机使用的 OLED 面板绝大多数属于 AMOLED；PMOLED 更常见于尺寸较小、显示内容较简单的设备。

三、OLED 的关键参数

3.1 亮度

参数 含义 旗舰水平
典型亮度 常规模式下的参考亮度 需结合厂商测试条件
峰值亮度 小窗口 HDR 高光或特定场景的瞬时亮度 不能代表全屏持续亮度
全屏亮度 高 APL（平均画面亮度）下的持续亮度 通常明显低于局部峰值

为什么峰值亮度比全屏亮度高？

因为 OLED 每个像素独立供电，局部高亮时电流集中，散热可控。全屏高亮时功耗和发热都会成为瓶颈。

3.2 PWM 调光

OLED 调节亮度有两种方式：

方式 原理 优缺点
DC 调光 直接改变电流大小 低亮度时色偏严重
PWM 调光 周期性调节发光时间占比 部分敏感用户在特定条件下可能不适

PWM 频率：

频闪体验不能只看频率，还要同时看调制深度、占空比、波形、屏幕亮度和个人敏感程度。较高频率通常有利于降低可感知频闪，但“1920Hz 就一定无感或安全”并不成立；类 DC 调光也可能在低亮度色准、灰阶或均匀性之间做取舍。

3.3 LTPO 技术

LTPO（Low-Temperature Polycrystalline Oxide）是一类结合低温多晶硅与氧化物 TFT 特性的背板方案，可降低低刷新率下的驱动功耗，并支持更宽的可变刷新率范围。1Hz~120Hz 是部分产品的实现，并非 LTPO 本身保证的固定范围。

场景 → 刷新率：
滑动列表 → 120Hz（流畅）
看静态图片 → 1Hz（省电）
播放视频 → 匹配帧率（24/30/60Hz）
息屏显示 → 1Hz（极低功耗）

市场上常见的“LTPO 2.0、3.0”等名称多为厂商营销口径，并不是统一行业标准。比较产品时，更应关注最低刷新率、切换策略、触控场景是否锁高刷，以及实测功耗。

3.4 排列方式

OLED 子像素的排列方式直接影响清晰度：

排列 特点 代表
RGB Stripe 标准排列，最清晰 LCD 屏幕
Pentile（钻石排列） 绿色像素多，红蓝共享 三星 AMOLED
类钻排列 各厂商变体 京东方、维信诺
Delta 排列 三角形排列 少数 OLED 面板

为什么 Pentile 排列争议大？

同样标称分辨率下，Pentile 排列的红、蓝子像素数量通常少于 RGB Stripe，细小文字和彩色边缘的观感可能不同。但不存在适用于所有面板和内容的固定“70%~80% 等效清晰度”，还要结合像素密度、渲染算法与观看距离判断。

四、OLED vs LCD

对比项 OLED LCD
发光方式 自发光 需要背光
黑色表现 纯黑（像素关闭） 灰黑（背光漏光）
对比度 暗室静态对比度极高 受面板类型与背光分区影响
厚度 更薄（无背光层） 较厚
可弯折 柔性基板产品可以弯折 常规手机 LCD 较难弯折
功耗 与画面内容和亮度强相关 主要受背光亮度影响
寿命 蓝色有机材料衰减快 相对更长
烧屏风险 有（长时间显示固定画面） 几乎没有
成本 较高 较低

五、OLED 的技术分支

5.1 柔性 OLED（Flexible OLED）

基板从玻璃换成聚酰亚胺（PI）塑料，可以弯折、卷曲。

应用：

  • 曲面屏（三星 Galaxy Edge 系列）
  • 折叠屏（Galaxy Z Fold、华为 Mate X）
  • 卷轴屏（概念产品）

5.2 Micro OLED / Micro LED

技术 特点 应用
Micro OLED 硅基 OLED（OLED-on-Silicon），超高 PPI Apple Vision Pro
Micro LED 无机自发光、亮度和寿命潜力高 量产与巨量转移难度高

Micro LED 被视为重要的下一代显示路线，但仍可能发生像素老化、亮度不均与坏点等问题；其成本也受尺寸、良率和制造方式影响，不能用固定的“OLED 十倍”概括。

六、屏幕供应商

供应商 特点 客户
三星显示（SDC） 中小尺寸 OLED 量产经验丰富 手机、穿戴与 IT 设备
LG Display（LGD） 大尺寸 OLED 与车载显示布局较深 电视、手机与车载设备
京东方（BOE） LCD 与柔性 OLED 产品线广 手机、IT 与车载设备
维信诺（Visionox） 柔性 OLED 与新型显示布局 手机与穿戴设备
天马微电子（Tianma） 中小尺寸显示覆盖面广 手机、车载与工业设备
华星光电（CSOT） 大尺寸显示与 OLED 持续投入 电视、手机与 IT 设备

"屏幕混用"现象：部分机型可能按批次采用不同供应商或不同版本的面板，但都应满足整机厂的规格要求。具体供应关系会随型号和批次变化，不宜仅凭品牌表格推断手中设备的面板。

七、选购建议

7.1 关键参数优先级

参数没有适用于所有人的固定优先级。户外使用者应先看持续高亮与反射率，内容创作者重视色准与色彩管理，游戏用户关注刷新率、响应时间和触控延迟，频闪敏感用户则应优先查看完整调光实测。

7.2 不同需求的侧重点

需求 重点关注
户外使用 关注高亮模式下的持续亮度、反射率与温控
频闪敏感 结合频率、调制深度和低亮度实测，最好亲自体验
看视频 DCI-P3 色域、HDR10+ / Dolby Vision
游戏 120Hz+、低触控延迟
折叠屏 关注折痕、UTG 玻璃厚度

八、常见问题

Q：OLED 烧屏怎么办？

A：避免在高亮度下长时间显示固定画面，使用自动亮度并让系统保持像素偏移、补偿等保护机制开启。烧屏取决于累计亮度、温度、内容与面板老化，无法承诺固定年限内一定不会发生。

Q：PWM 调光真的伤眼吗？

A：频闪可能令部分敏感用户在低亮度下感到眼疲劳、头痛或不适，但不能用单一频率判断影响。选购时应查看频率与调制深度的完整实测，并亲自测试常用亮度；“护眼模式”主要调节色温，也不等同于消除频闪。

Q：2K 屏幕比 1080P 耗电多少？

A：没有固定比例。更高分辨率会增加 GPU 渲染和部分显示链路负担，但实际差异还取决于应用是否原生渲染、刷新率、面板、亮度和 SoC。应以具体机型在相同条件下的续航测试为准。

参考资料：三星显示技术白皮书、DisplayMate 屏幕评测、DXOMARK 显示评分标准
